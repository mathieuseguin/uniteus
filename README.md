# Goal

This program prints out a multiplication table of the first 10 primary numbers.


# Install

```
$ bundle install
````


# Run tests

```
$ rspec
```


# Usage

```
$ ./bin/multiply_primes 3
```


# Example
```
---------------------------------
|       |     2 |     3 |     5 |
---------------------------------
|     2 |     4 |     6 |    10 |
---------------------------------
|     3 |     6 |     9 |    15 |
---------------------------------
|     5 |    10 |    15 |    25 |
---------------------------------
```
